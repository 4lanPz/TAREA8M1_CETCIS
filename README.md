# Sistema de Gestión de Inventario y Pedidos

## Descripción

Este programa es un sistema básico desarrollado en **Python** que permite gestionar un **inventario de productos** y **pedidos de clientes** desde la consola.  

El sistema permite registrar, visualizar, editar y eliminar información utilizando funciones para mantener el código organizado y fácil de ampliar.

Para realizar ese reto se usaron conceptos básicos como:

- Entrada y salida de datos
- Uso de funciones
- Estructuras condicionales
- Ciclos
- Funciones y procedimientos
- Listas y diccionarios para almacenar información

---

# Funcionamiento del sistema

El programa funciona mediante un **menú principal** desde el cual el usuario puede elegir entre:

1. Gestionar inventario  
2. Gestionar pedidos  
3. Salir del sistema  

Cada opción abre un **submenú** con diferentes acciones que permiten manipular la información del sistema.

---

# Variables principales

### `inventario`

Es una **lista** que almacena todos los productos registrados.

Cada producto se guarda como un **diccionario** con la siguiente estructura:

```python
{
    "nombre": "Producto",
    "cantidad": 10,
    "precio": 25.5
}
```
